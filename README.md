# Socket-python

# Setup for 1 machine
## Setup for MacOS/Linux
```
git clone https://github.com/linhnph05/Socket-python
cd Socket-python
python3 -m venv venv 
source venv/bin/activate
```

### Client
```
python3 client.py
```

### Server
```
python3 server.py
```

## Setup for Windows

```
git clone https://github.com/linhnph05/Socket-python
cd Socket-python
python3 -m venv venv 
.\venv\Scripts\activate
```

### Client
```
python3 client.py
```

### Server
```
python3 server.py
```
