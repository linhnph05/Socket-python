# Socket-python

# Setup for 1 machine
## Setup for MacOS/Linux
```
git clone https://github.com/linhnph05/Socket-python
cd Socket-python
python3 -m venv venv 
source venv/bin/activate
pip3 install -r requirements.txt
```

### Client
```
cd client
python3 client.py
```

### Server
```
cd server
python3 server.py
```
